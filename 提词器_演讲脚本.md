## 第 1 页：封面（10 秒）

> 大家好，我的课设题目是「图像超分辨率——基于卷积神经网络的图像重建」。我实现了两种模型：SRCNN 和 SRResNet，做了对比分析。2 倍放大，SRCNN 达到 47.99 dB PSNR，SRResNet 有 56 万参数。

---

## 第 2 页：项目简介（45 秒）

> 图像超分辨率的任务是把一张低分辨率的小图放大到高分辨率。传统 Bicubic 插值放大后边缘模糊、纹理丢失。
>
> 我用深度学习来解决：先训练 SRCNN 作为基线，再改进为深层残差网络 SRResNet，对比两者的重建质量。训练是自监督的，不需要人工标注：把原图下采样得到低分辨率版本，用模型去恢复它，和原图对比计算损失。

**专有名词**：

| 名词 | 解释 |
|------|------|
| **Bicubic 插值** | 双三次插值，传统放大方法，用周围 16 个像素做三次多项式拟合估算新像素值。速度快但无法恢复高频细节 |
| **SRCNN** | Super-Resolution CNN，2014 年提出，第一个用卷积神经网络做超分辨率的工作，3 层卷积就超越了传统方法 |
| **SRResNet** | Super-Resolution Residual Network，加入残差连接和批归一化，可堆叠更深而不退化 |
| **自监督学习** | 不需要人工标注的学习方式。用原图自身生成低分辨率图作为输入，原图作为标签 |

---

## 第 3 页：数据集介绍（40 秒）

> 训练集用了 397 张自然场景图片，场景多样、纹理丰富。验证集用 Set14 标准测试集 14 张。数据是自监督的，无需标注。
>
> 没下载标准的 DIV2K 数据集，因为它托管在海外服务器，3.5GB 下载极慢。我直接复用本地已有的自然图片，零额外下载。还加了回退机制：找不到本地图时自动生成合成训练数据，确保任何环境都能训练。
>
> 处理流程：从原图随机裁剪 192px 高分辨率块，Bicubic 下采样到 96px，再放大回 192px 作为模型输入。每张图取 30 个随机位置，加上随机翻转和旋转做数据增强。

**专有名词**：

| 名词 | 解释 |
|------|------|
| **DIV2K** | 超分标准数据集，800 张 2K 分辨率高质量图片。本项目因网络问题未使用 |
| **Epoch** | 模型完整遍历一遍全部训练数据。这里每 epoch 约 744 个 batch |
| **Batch** | 批量大小，每次更新参数用的样本数，设为 16 |
| **Adam** | 自适应学习率优化算法，结合动量和 RMSProp 优点，收敛快且稳定 |
| **CosineAnnealing** | 余弦退火学习率调度：学习率按余弦曲线从初始值平滑衰减到接近零 |

---

## 第 4 页：模型架构（55 秒）

> SRCNN 非常简单：三层卷积。第一层 9×9 大卷积核提取图像块特征，第二层 1×1 做非线性映射，第三层 5×5 重建输出。只有 57K 参数，训练极快。
>
> SRResNet 是深层残差网络：3×3 卷积输入后堆叠 8 个残差块，每个残差块是 Conv→BN→ReLU→Conv→BN，最后加跳跃连接 Output=F(x)+x。还有全局跳跃连接把第一层输出加到中间层。总计 560K 参数。
>
> 公平对比：两个模型输入完全相同，都是 Bicubic 放大后的低分辨率图像，只对比网络结构差异。

**专有名词**：

| 名词 | 解释 |
|------|------|
| **卷积核（Kernel）** | 卷积层的滑动窗口。9×9 感受野大但参数多；3×3 感受野小但可堆更多层 |
| **Residual Block（残差块）** | 核心公式 Output = F(x) + x。把输入 x 直接加到卷积输出上，梯度可直通浅层，解决深层网络退化问题 |
| **BatchNorm（批归一化）** | 对每个 mini-batch 做标准化（均值 0，方差 1），稳定训练，加速收敛 |
| **Global Skip Connection** | 网络浅层输出直接加到深层中间层。极端情况下网络至少可输出恒等映射，不会退化 |
| **ReLU** | 激活函数 f(x)=max(0,x)。大于 0 原样通过，小于 0 截断为 0。非线性，计算极快 |

## 第 6 页：实验结果（40 秒）

> 定量来看，SRCNN PSNR 47.99dB，SRResNet 47.75dB。但 PSNR 只是像素级指标，不代表人眼感知。视觉上 SRResNet 纹理更清晰、边缘更锐利。
>
> 对比面板展示从低分辨率到 Bicubic、SRCNN、SRResNet 再到原始高清的逐级提升。最直观的是细节放大区域：Bicubic 模糊、SRCNN 可辨识、SRResNet 接近原图。训练曲线显示 SRCNN 快速收敛，SRResNet 前期波动但最终超越。

**专有名词**：

| 名词 | 解释 |
|------|------|
| **PSNR（峰值信噪比）** | Peak Signal-to-Noise Ratio，图像质量指标。公式 PSNR = 20×log₁₀(MAX/√MSE)。>40dB 算优秀。但只看像素差异，不考虑人眼感知 |
| **MSE（均方误差）** | 预测值和真实值之间差值的平方和的均值，训练时的损失函数 |
| **Ground Truth** | 真实值/标签。超分中指原始高分辨率图像，模型输出的参照标准 |

## 第 7 页：核心工作（65 秒，最重要）

> 核心工作围绕三个瓶颈展开。
>
> 第一个瓶颈：SRCNN 太浅，纹理有上限。我修改 model.py，引入残差块和批归一化，堆 8 个残差块，加全局跳跃连接。效果是 SRResNet 纹理明显优于基线，高频区域边缘更锐利。
>
> 第二个瓶颈：SRResNet 训练初期不稳定。我加了 CosineAnnealing 学习率衰减和数据增强。效果是训练曲线平滑，PSNR 稳步收敛到 47.75dB，自动保存最佳模型。
>
> 第三个瓶颈：标准数据集从海外下载太慢。我改 dataset.py，复用本地已有图片，自动生成训练对。效果是零额外下载，训练流畅。还加了回退机制：找不到图就自动生成合成数据。

**专有名词**：

| 名词 | 解释 |
|------|------|
| **梯度（Gradient）** | 损失函数对每个参数的偏导数，指示参数调整方向。反向传播自动计算 |
| **梯度消失** | 深层网络中梯度逐层衰减到零，浅层参数几乎不更新。残差连接提供「高速公路」让梯度直通 |
| **学习率（Learning Rate）** | 控制每次参数更新的步长。太大跳过最优解，太小收敛太慢 |
| **正则化（Regularization）** | 防止过拟合的技术。weight_decay 是 L2 正则化，限制参数不要太大 |
